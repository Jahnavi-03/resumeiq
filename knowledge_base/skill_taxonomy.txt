TECH SKILL TAXONOMY BY ROLE
==============================
This taxonomy defines required and preferred skills
for each role. Use this to identify skill gaps when
comparing resume against job description.

SECTION 1: DATA ENGINEER SKILLS
==============================

TIER 1 — CORE REQUIRED (missing any = major gap):
Programming:
  - Python (advanced: not just scripting)
  - SQL (complex queries, window functions, CTEs,
    query optimization)

Distributed Computing:
  - Apache Spark
  - PySpark

Version Control:
  - Git
  - GitHub or GitLab

TIER 2 — PLATFORM REQUIRED (need at least one):
Cloud Platforms (pick one):
  - Azure (Azure Data Factory, Azure Databricks,
    Azure Data Lake, Azure Synapse)
  - AWS (S3, Glue, EMR, Redshift, Lambda)
  - GCP (BigQuery, Dataflow, Pub/Sub, Composer)

Data Warehouses (pick one):
  - Databricks
  - Snowflake
  - BigQuery
  - Amazon Redshift
  - Azure Synapse Analytics

Orchestration (pick one):
  - Apache Airflow
  - Azure Data Factory
  - Prefect
  - Dagster
  - AWS Step Functions

TIER 3 — STRONG DIFFERENTIATORS:
Streaming:
  - Apache Kafka
  - Spark Streaming
  - Azure Event Hubs
  - AWS Kinesis
  - Google Pub/Sub

Storage Formats:
  - Delta Lake
  - Apache Iceberg
  - Apache Parquet
  - Apache Avro

Infrastructure as Code:
  - Terraform
  - Databricks Asset Bundles
  - AWS CDK
  - Azure Bicep

  - Great Expectations
  - dbt (data build tool)
  - Monte Carlo

TIER 4 — NICE TO HAVE:

SECTION 5: HOW TO USE THIS TAXONOMY
When analyzing a resume against a JD:

Step 1: Identify the target role from JD
Step 2: Look up Tier 1 skills for that role
Step 3: Check if ALL Tier 1 skills are in resume
Step 4: Check Tier 2 — at least one from each category?
Step 5: Note which Tier 3 skills are present as strengths
Step 6: Note which Tier 3 skills are missing as gaps
Step 7: Use JD to identify which specific skills matter most

Reporting skill gaps:
- Tier 1 missing = CRITICAL gap (mention first)
- Tier 2 missing = MAJOR gap (mention second)  
- Tier 3 missing = MINOR gap (mention if relevant to JD)
- Tier 4 missing = OPTIONAL (only mention if JD asks for it)
